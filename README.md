## Projects

### 1. Computational Toxicology Prediction
This project focuses on predicting **Drug-Induced Liver Injury (DILI)** using a KNIME-based computational toxicology workflow. Drug-induced liver injury is one of the leading causes of drug development failure and post-marketing withdrawal, making early toxicity prediction essential.

The workflow uses molecular structure data represented in SMILES format, processed through RDKit within KNIME, to generate molecular fingerprints for toxicity prediction. A Random Forest classification model is implemented to classify compounds as toxic or non-toxic.

**Key components:**
- Chemical structure preprocessing
- Molecular fingerprint generation
- Random Forest classification
- Cross-validation
- ROC curve analysis
- Confusion matrix evaluation

**Technologies used:**
- KNIME Analytics Platform
- RDKit
- Random Forest
- SMILES molecular representation

**Applications:**
- Computational toxicology
- Drug safety screening
- Early-stage toxicity prediction

### 2. Drug Repurposing
This project explores computational drug repurposing to identify existing compounds with potential therapeutic applications for alternative disease indications. Drug repurposing offers a cost-effective strategy by leveraging known compounds with established safety profiles.

The project applies cheminformatics preprocessing, molecular descriptor generation, and predictive modelling workflows to identify and prioritise candidate compounds.

**Key components:**
- Dataset preparation
- Molecular preprocessing
- Descriptor and fingerprint generation
- Machine learning-based compound prioritisation
- Candidate screening

**Technologies used:**
- KNIME
- RDKit
- Python
- Scikit-learn
- Molecular fingerprinting

**Applications:**
- Drug repositioning
- Candidate therapeutic discovery
- Computational pharmacology

### 3. Natural Products in Drug Design
This project focuses on computational analysis of natural products as potential drug candidates. Natural compounds are an important source of pharmacologically active molecules due to their structural diversity and biological relevance.

The workflow evaluates molecular properties, structural characteristics, and drug-likeness of natural compounds using cheminformatics tools.

**Key components:**
- Natural compound dataset analysis
- Molecular descriptor calculation
- Drug-likeness assessment
- Structural property evaluation
- Compound filtering

**Technologies used:**
- KNIME
- RDKit
- Python
- Molecular descriptor analysis

**Applications:**
- Natural product drug discovery
- Lead compound identification
- Bioactive compound screening
