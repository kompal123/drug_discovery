# Drug Discovery Portfolio

## Overview
This repository contains a collection of computational drug discovery projects demonstrating the application of **cheminformatics, machine learning, computational toxicology, and drug repurposing** in modern pharmaceutical research. The projects showcase practical workflows developed using computational tools such as **KNIME, RDKit, Python, and machine learning models** to support safer, faster, and more efficient drug discovery.

The repository is organised into three independent project areas, each focusing on a different aspect of computational drug discovery.
